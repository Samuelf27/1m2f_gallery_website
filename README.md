# 🎨 1M2F Art Gallery Website

Plataforma **full stack** de site de galeria de arte desenvolvida como parte do portfólio da 1M2F, com foco em **experiência do usuário, design responsivo e integração entre front‑end e back‑end**.

---

## 🚀 Sobre o Projeto

Este projeto consiste na recriação do site da 1M2F, transformando‑o em uma aplicação web profissional que exibe obras de arte de forma visualmente atraente. A galeria de projetos atua como uma **vitrine interativa** para trabalhos artísticos, permitindo ao usuário navegar facilmente pelos conteúdos, com layout responsivo para todos os tamanhos de tela. :contentReference[oaicite:1]{index=1}
